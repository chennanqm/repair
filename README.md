本次实习以“手机维修管理系统”为实战载体，一次性拉起“SpringBoot 后端 + MySQL 数据库 + Vue 前端”三套环境，形成标准的前后端分离工作流，为后续功能迭代备好跑道。
环境一：SpringBoot 后端
① IntelliJ IDEA：项目脚手架、代码编写、调试一条龙；
② Maven：一键拉取 Jar 并统一版本，杜绝依赖冲突；
③ JDK 17：拥抱 SpringBoot 最新特性，性能语法双提升。
环境二：数据持久层
① MySQL 8：关系型数据库，工单、配件、用户结构化落地；
② Navicat Premium：图形化客户端，建库建表、SQL 调试、数据导入导出可视化完成。
环境三：Vue 前端
① Node.js 18：自带 npm 包管理器，为工程化构建提供运行时；
② Vue 3 + Vite：响应式框架加持热更新，页面渲染与交互开发效率翻倍。
三套环境彼此解耦又通过 REST 接口无缝协同，保存一次脚本即可全局复现，让项目从 0 到 1 的运行支撑稳若磐石。
